# Blogup
Blogup is a full-stack MERN blog app. User can register, create, and delete a blog.

<h2>Technologyies Used</h2>
  <ul>
    <li>Typescript</li>
    <li>React.Js</li>
    <li>Mongo DB</li>
    <li>Express</li>
    <li>Node.Js</li>
  </ul>
  
  Demo:- https://blogup-delta.vercel.app/
